Success criterion. A cell turns green ✓ when the agent's discovered circuit cost is at most the minimum cost across all five closed-form baselines (sparse-direct, Horner, multivariate Horner, CSE, and top-down branch-and-bound). Cells turn amber ✗ when the agent fails to match any of those baselines.

Modes. Greedy rolls out the policy alone — one forward pass per frontier step, taking the argmax candidate. Search wraps the same network in a 32-simulation memoized AND/OR tree search with PUCT selection. Hover any cell for the underlying numbers.
